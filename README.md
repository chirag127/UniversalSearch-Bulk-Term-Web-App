# MultiSearchX

MultiSearchX is a web application that allows users to search multiple terms at once across various search engines, and even includes an option to append a sentence to each search term before conducting the search.

## Usage

To use MultiSearchX, simply navigate to <http://chirag127.github.io/MultiSearchX/> in your web browser. Enter your search terms into the provided text area, and optionally enter text to append to each search term. Then, select a search engine from the drop-down menu, and click the "Search" button. MultiSearchX will open a new tab for each search term, using the selected search engine and appended text if provided.

## Features

1. **Search multiple terms at once**: Users can enter multiple search terms separated by a new line and the website will open a new tab for each search term in the selected search engine.

2. **Option to append/prepend text**: Users can choose to add custom text before or after each search term to refine their search.

3. **Multiple search engines**: Users can choose from various popular search engines organized into categories (General, AI-Powered, and Custom).

4. **Collapsible categories**: Search engines are organized into collapsible sections for better organization and easier navigation.

5. **Search counter**: Real-time display of the total number of searches that will be performed based on selected options.

6. **Simple and user-friendly interface**: The website has a clean and easy-to-use interface, making it accessible for users of all skill levels.

7. **Customizable search experience**: Users can choose their preferred search engines, customize their search terms, and configure delays between searches.

8. **Open source**: The code for MultiSearchX is available on GitHub, allowing developers to contribute to and modify the project.

9. **Responsive design**: The website is optimized for different screen sizes, making it easy to use on desktop and mobile devices.

## Supported Search Engines

MultiSearchX currently supports the following search engines:

### General Search Engines
- **Google** - The world's most popular search engine
- **Google I'm Feeling Lucky** - Goes directly to the top search result
- **DuckDuckGo** - Privacy-focused search engine
- **DuckDuckGo Lucky** - Goes directly to the top result on DuckDuckGo

### AI-Powered Search Engines
- **Bing** - Microsoft's search engine with AI capabilities
- **Phind** - AI search engine optimized for developers
- **Perplexity** - AI-powered answer engine
- **Neeva AI** - Ad-free AI search engine
- **Brave** - Privacy-focused search with AI features
- **YouChat** - AI conversational search from You.com

- Naver

- Google Images
- Google Videos
- Google News
- Google Shopping
- Google Books

## Advanced Features

- **Bulk operations**: Select all engines or all AI engines with a single click
- **Custom delimiters**: Define custom separators for your search terms
- **Search delays**: Set delays between searches to avoid overwhelming your browser
- **Multiple searches per term**: Optionally perform multiple searches for each term
